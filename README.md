# MovieStudio

MovieStudio (Movieholic) is an Android application that displays movie information using the TMDB API.  
The app allows users to explore movies by genre, search for movies, view detailed information, read reviews, and watch trailers.

---

## Features

- 🎬 Genre-based movie filtering
- 🔍 Movie search functionality
- ♾️ Pagination (endless scrolling) for movies and user reviews
- ⭐ Detailed movie information with ratings
- ▶️ YouTube trailer integration

---

## Tech Stack

| Category | Technology |
|--------|-------------|
| Language | Kotlin |
| UI Framework | Jetpack Compose (Material 3) |
| Architecture | MVVM + Clean Architecture (Domain, Data, Presentation) |
| Dependency Injection | Dagger Hilt |
| Networking | Retrofit2 + OkHttp3 |
| Image Loading | Coil |
| Async Handling | Kotlin Coroutines & Flow |
| Navigation | Jetpack Navigation Compose |
